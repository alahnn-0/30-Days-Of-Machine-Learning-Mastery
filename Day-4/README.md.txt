# Day 4 - Linear Regression & Model Evaluation

## Topics Covered

### 1. Linear Regression

Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values by finding the best-fit line through data points.

### Linear Regression Equation

y = mx + b

Where:
- y = Predicted value
- m = Slope
- x = Input feature
- b = Intercept

---

### 2. Fitting a Line to Data

The goal of Linear Regression is to find the line that minimizes the prediction error between actual and predicted values.

A good regression line should:
- Capture the overall trend
- Minimize prediction errors
- Generalize well to unseen data

---

### 3. Performance Measures

Performance measures help evaluate how well a machine learning model performs.

#### Mean Absolute Error (MAE)

MAE = Σ|Actual - Predicted| / n

Measures the average absolute error.

#### Mean Squared Error (MSE)

MSE = Σ(Actual - Predicted)² / n

Penalizes larger errors more heavily.

#### Root Mean Squared Error (RMSE)

RMSE = √MSE

Provides error in the same unit as the target variable.

---

## Key Takeaways

✅ Linear Regression predicts continuous values.

✅ The best-fit line minimizes prediction errors.

✅ MAE, MSE, and RMSE are common evaluation metrics.

✅ Lower error values generally indicate better model performance.
