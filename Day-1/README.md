## 📚 Topics Covered

### 1. Introduction to Machine Learning

Machine Learning (ML) is a branch of Artificial Intelligence that enables computers to learn patterns from data and make predictions or decisions without being explicitly programmed.

#### Types of Machine Learning
**Youtube : **
Indroduction to ML : https://youtu.be/Gv9_4yMHFhI?si=7rDnezZ74OfRTKrI
                      https://youtu.be/EuBBz3bI-aA?si=63zeU-N0ZXZs7PRr
- **Supervised Learning**
  - Learns from labeled data.
  - Examples:
    - House Price Prediction
    - Student Performance Prediction

- **Unsupervised Learning**
  - Learns from unlabeled data.
  - Examples:
    - Customer Segmentation
    - Clustering

- **Reinforcement Learning**
  - Learns through rewards and penalties.
  - Examples:
    - Game Playing AI
    - Self-driving Systems

### 2. NumPy Basics

NumPy (Numerical Python) is a powerful Python library used for numerical computations and array operations.

#### Why NumPy?

- Faster than Python lists
- Efficient memory usage
- Supports mathematical operations
- Foundation for Data Science and Machine Learning libraries

#### Topics Learned

- Creating Arrays
- Array Dimensions
- Array Shape
- Indexing and Slicing
- Mathematical Operations
- Basic Statistics Functions

---

## 💻 Practice Code

### Creating Arrays

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])

print(arr)
```

### Basic Operations

```python
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

print(a + b)
print(a * b)
```

### Statistical Functions

```python
import numpy as np

arr = np.array([10, 20, 30, 40, 50])

print(np.mean(arr))
print(np.max(arr))
print(np.min(arr))
```
