## 🎯 Objective

This repository documents my journey of learning **Machine Learning from scratch** over the next 30 days.

The goal is not just to learn machine learning algorithms, but to understand the complete data workflow—from collecting and analyzing data to building, evaluating, and improving models.

Throughout this challenge, I will:

* Learn new concepts every day.
* Practice by writing code.
* Build projects to apply my knowledge.
* Document everything I learn.
* Track my progress publicly.
